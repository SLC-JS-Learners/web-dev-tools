Accessing

We'll start by accessing your last toolview or the inspector

  • OS X: Cmd + Opt + I
  • Firefox:
    • Right-click on any page element and select Inspect Element.
    • From the Firefox menubar: Tools > Web Developer > Inspector
    • Windows and Linux: Ctrl + Shift + I
    • Additional Firefox shortcuts https://developer.mozilla.org/en-US/docs/Tools/Keyboard_shortcuts
  • Chrome:
    • Right-click on any page element and select Inspect.
    • Open the Chrome menu (hamburger icon) at the top-right of your browser window, then select Tools > Developer Tools.
    • Windows: F12, Ctrl + Shift + I
    • Additional Chrome shortcuts https://developers.google.com/web/tools/chrome-devtools/iterate/inspect-styles/shortcuts
